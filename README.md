# 🧠 Exercícios de Lógica de Programação

Repositório com exercícios voltados ao desenvolvimento da lógica de programação.

## 🚀 Objetivo
Fortalecer o raciocínio lógico e a resolução de problemas utilizando programação.

## 🛠️ Tecnologias utilizadas
- JavaScript

## 📌 Conteúdo
- Estruturas condicionais (if/else)
- Laços de repetição
- Validação de entrada
- Resolução de problemas lógicos

## 📚 Aprendizados
- Pensamento lógico estruturado
- Organização de soluções
- Escrita de código mais clara e eficiente

## ⚠️ Observação
Os exercícios aqui representam minha evolução contínua e prática diária de lógica de programação.
